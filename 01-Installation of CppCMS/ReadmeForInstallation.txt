

1. Visit this site :--

	http://cppcms.com/wikipp/en/page/cppcms_1x_build


2. install following packages for debian:--
	sudo apt-get install cmake libpcre3-dev zlib1g-dev libgcrypt20 libicu-dev python git

3. Then clone from git repo.
	
	git clone https://github.com/artyom-beilis/cppcms.git cppcms 

4. cd to cppcms directory.

5. mkdir build inside cppcms directory.

6. cd build.

7. cmake ..

8. cmake various_build_options ..  

9. make

10. make test

11. make install
